# 成績整頓君

CSVで出力された成績データをアップロードし、
時間割コードおよび開講年度に基づいて単位区分ごとに単位数を付与・合計単位数を自動計算するWebアプリです。  
現在、2023年以降の入学者で下記の学部生に対応しています。
- 総合理工学部
- 材料エネルギー学部

※本アプリは公開情報をもとに作成された個人開発ツールであり、公式機関とは一切関係ありません。

---

## 📌アプリ概要

本アプリは以下の流れで動作します。

1. 学部・入学年度を選択
2. ユーザーがCSVをアップロード
3. 事前取得したシラバス情報（JSON）を参照
   - 時間割コードで科目区分を取得
   - 時間割コード＋年度で単位数を一意に取得
4. 単位区分ごとに集計し、結果を出力

マッチングは時間割コードおよび開講年度で一意に決定されるため、
曖昧検索は行っていません。
